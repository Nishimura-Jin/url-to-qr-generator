# 注釈付きQRコード生成ツール

URLを入力するだけで、QRコードとその上下に「日本語の注釈」を付与した画像を生成・ダウンロードできるWebアプリです。

## 🚀 公開中のアプリ
[https://url-to-qr-generator-jin.streamlit.app/](https://url-to-qr-generator-jin.streamlit.app/)

## 💡 開発の背景と想い
食品のラベルや書籍など、日常生活のあらゆる場面で信頼され活用されている「QRコード」という技術に深い関心を持ち、 **「この社会に不可欠な仕組みを、自分の力（独学）でゼロから構築してみたい」** という探究心から開発をスタートしました。

単にプログラム上でQRコードを生成して完結させるのではなく、開発の過程で **「Web上の表示で終わらせず、名刺や掲示物として実際に印刷して活用することで、より現実社会での利便性が高まる」** という可能性に気づき、プロダクトとしての実用性を追求しました。

### ✨ 実用性と信頼性を高めるためのこだわり
既存のQRコード生成ツールの多くは、コードのみを出力するものが大半です。しかし、実務での運用を想定し、以下の2点にこだわって実装しました。

- ・**「注釈（テキスト）付与」によるホスピタリティ**
    - ・**情報の即時性**: ユーザーがスマホをかざす前に、注釈によって「何のためのコードか」を理解できる工夫を施しました。
    - ・**物理的な実用性**: 印刷して運用する際、複数のコードを並べても内容を混同せず、貼り間違いなどのミスを防ぐことができます。

- ・ **「検証用リンク」によるミス防止（UXの向上）**
    - ・**スイッチングコストの削減**: PC作業中にわざわざスマホを取り出して読み取りテストをする手間を省き、画面上のボタン一つでリンク先を確認できる機能を実装しました。
    - ・**リスクマネジメント**: 「印刷した後にURLの間違いに気づく」という致命的なミスを、生成直後のワンクリックで防ぐという実務的な配慮を込めています。
    
「自分が作ったものを、名刺や資料に載せて即座に人に手渡せる」。そんな **「物理的な手触り感のある技術証明」** が、本アプリの最大の特徴です。

## 🛠 使用技術
- **Language**: Python 3.x
- **Framework**: Streamlit
- **Libraries**: 
  - `qrcode` (QRコード生成)
  - `Pillow` (画像処理・日本語描画)
- **Infrastructure**: Streamlit Cloud

## 📝 使い方
1. **URLの入力:** 生成したいURLを入力欄に貼り付けます。

2. **注釈の設定:** QRコードの上下に表示したいテキスト（注釈）を入力します。

3. **内容の検証:**「QRコードを生成」ボタンを押し、プレビューを確認して画像をダウンロードしてください。

4. **保存:** プレビューに問題がなければ、画像をダウンロードして活用してください。
